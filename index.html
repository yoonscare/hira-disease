HIRA 한방 질병 조회 건강보험심사평가원 질병정보서비스
처음 사용하시나요?
  1. www.data.go.kr 접속 → 회원가입
  2. "건강보험심사평가원" 검색 → 질병정보서비스 신청 (자동승인)
  3. 마이페이지 → 개발계정 → 일반 인증키 복사
  4. 아래에 인증키 입력 후 질병명 검색
인증키 입력

인증키는 브라우저에만 저장되며 외부로 전송되지 않습니다.
